== Beschreibung ==

Das Plugin Petfinder ist eine Erweiterung des kostenpflichtigen Pet-Manager.
Der Pet-Manager ist direkt aus der Praxis entstanden und wird laufend weiterentwickelt. Durch die langjährigen Erfahrungen und den regelmäßigen Kontakt zu unseren Kunden kennen wir deren Wünsche und Anforderungen an unsere Software. Dies alles fließt in die Weiterentwicklung mit ein. 
Um jedem Verein eine günstige Lösung bieten zu können, ist der pet-Manager modular aufgebaut. Ausgehend von der Grundversion, die bereits alle wichtigen Funktionen zur Verwaltung Ihrer Tiere bietet, können Sie den Pet-Manager mit weiteren Modulen erweitern. So kann jeder Verein sein individuelles Paket buchen.

Mit dem Modul Homepage-Management und diesem Plugin binden Sie Ihre Tiere aus dem Pet-Manager komfortabel in Ihre Homepage ein. 

Stellen Sie Listen nach Geschlecht, Tierart usw. zusammen und lassen Sie den Inhalt direkt vom Pet-Manager verwalten. So werden z.B. vermittelte Tiere automatisch in die Liste der bereits vermittelten Tiere verschoben. Wenn ein Tier verstorben ist, wird es automatisch in die Liste der verstorbenen Tiere (Regenbogenbrücke) verschoben etc.

In Verbindung mit dem Tierheim-Management können zusätzlich die Fundtiere vollautomatisch auf der Homepage angezeigt werden.


= Funktionen =
* Auswahl verschiedener Listen (vermittelbar, vermittelt, verstorben, Fundtier, Fremdvermittlung, Handicaptiere, Notfälle, Welpen/Kitten, Senioren, Patengesuch)
* UND Verknüpfungen für Tierarten und Aufenthaltsorte möglich
* Zufallsanzeige der Tiere einer bestimmten Auswahl (z.B. für die Startseite)
* Einbindung in beliebigen Seiten / Beiträgen per Shortcode
* Filtermöglichkeiten für den Nutzer nach Tierart, Geschlecht, Aufenthaltsort, Kategorie und Eigenschaften (konfigurierbar/abschaltbar)
* Detailansicht mit Bildergalerie
* Videoeinbindung durch Embedcode
* Krankheiten der Tiere anzeigen (konfigurierbar im Pet-Manager)
* Impfungen der Tiere anzeigen (konfigurierbar im Pet-Manager)
* Eigenschaften der Tiere anzeigen (konfigurierbar im Pet-Manager)
* Verschiedene Layouts wählbar
* Kontaktmöglichkeiten wählbar zwischen Betreuer des Tieres, Pflegestelle, manuelle Angaben, internes Kontaktformular oder externes Kontaktformular
* Separate Formulare für Adoptions- und Patenschaftsanfragen
* Unterstützung von [Contact Form 7](https://de.wordpress.org/plugins/contact-form-7/) (Direkte Übergabe der Tier- und Empfängerdaten)
* Bei Nutzung des internen Formulars direkte Speicherung im Pet-Manager möglich
* Individuelle Anzeige aller verfügbaren Felder. Jedes Feld kann einzeln sichtbar/unsichtbar geschaltet werden
* Patenschaftsbedarf und bereits erhaltene Patenschaften anzeigbar
* Vorhandene Paten direkt beim Tier anzeigen lassen
* Patenliste als separate Liste möglich

== Frequently Asked Questions ==

= Kann ich mit diesem Plugin meine Tiere online verwalten? =

Dieses Plugin nutzt die Daten aus dem [Pet-Manager](https://www.pet-man.eu/) und kann nicht alleine verwendet werden. Damit Sie dieses Plugin nutzen können, muss zum Pet-Manager das Modul Homepage-Management gebucht werden. Dies ist mit weiteren Kosten verbunden.

= Ist die Nutzung des Plugins kostenlos? =

Für die Nutzung des Plugins ist das Modul Homepage-Management als kostenpflichtige Erweiterung zum Pet-Manager Voraussetzung. Die Nutzung des Plugins selbst ist kostenlos.

= Wo erhalte ich Support? =

Im [Onlinehandbuch](https://helpcenter.kw-management.de/portal/de/kb/articles/installation-des-wordpress-plugins) finden Sie viele nützliche Anleitungen und Tipps. Zusätzlich erhält jeder Kunde des Pet-Manager ein Freikontingent und Zugang zum Ticketsystem. Supportanfragen werden innerhalb des vereinbarten Servicelevels bearbeitet.
Ein Support hier auf WordPress.org findet nicht statt!

= Kann ich den Pet-Manager testen? =

Sie können den Pet-manager 30 Tage kostenlos und ohne Verpflichtung testen! Weitere infos finden Sie auf [https://www.pet-man.eu/](https://www.pet-man.eu/)

= Wie können Daten an ein externes Formular übergeben werden? =

Bei Einsatz des Plugins [Contact Form 7](https://de.wordpress.org/plugins/contact-form-7/) können verschiedene Daten automatisch an das Formular übergeben werden. Hierzu gehören neben dem Tiernamen auch die ID, die laufende Nummer, die interne Bezeichnung, die E-Mail des Empfängers und der Aufenthaltsort. Eine genaue Anleitung finden Sie im [Onlinehandbuch](https://helpcenter.kw-management.de/portal/de/kb/articles/%C3%BCbergabe-an-contact-form-7).

= Kann das Aussehen der Listen individuell angepasst werden? =

Neben den eingebauten Konfigurationsmöglichkeiten für die Listen- und der Detailansicht, kann das gesamte Aussehen durch individuelle CSS Anpassungen geändert werden. Eine Anleitung hierzu finden Sie im [Onlinehandbuch](https://helpcenter.kw-management.de/portal/de/kb/articles/individuelle-gestaltung-der-anzeige)
